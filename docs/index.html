Size

온라인 쇼핑몰에서 바지를 구매할 때 사이즈 선택 실패로 힘들어하는 사용자를 위해
사이즈를 측정 후 사이즈를 추천하고 추천된 사이즈가 있는 바지를 추천해
온라인 쇼핑을 더욱 편리하게 하도록 해주는 앱.

주요 기능

  • 사이즈 측정

    ARKit 활용

  • 사이즈 추천

    coreML Tabular Classifier

  • 옷장

    realm db 이용해서 데이터 저장

세부 기능

  • 사이즈 측정

바지 사이즈를 측정하기 위해 아이폰 기본앱인 측정앱에서 사용하는
ARKit 를 사용하여 시작지점과 끝지점을 카메라로 측정하여 바지의
허리,허벅지,밑단,총장에 대한 길이 정보를 저장.

  • 사이즈 추천

쇼핑몰의 옷의 길이에 대한 정보들을 csv파일로 저장하여 필요한 데이터셋을 확보 한 후 CoreML Tabular Classifier 를 사용 한 후 허리 , 허벅지 , 밑단 , 총장을 feature 값으로 설정하고 예측하려고 하는 size를 target으로 설정해 학습한 모델을 통해 입력받은 옷의 정보와 가장 비슷한 사이즈를 찾아낼 수 있으며, 이 사이즈를 필터로 같은 사이즈의 옷들만 검색하여 쇼핑할 수 있도록 함.
